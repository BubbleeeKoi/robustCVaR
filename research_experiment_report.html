1. 研究背景与代码做了什么

1.1 理论来源

本项目基于 Ni & Lai (2024) 的 Robust Risk-Sensitive RL with CVaR。原文在有限 GridWorld RMDP 中研究：

\[ \min_\pi \max_{\tilde P\in\mathcal P} \mathrm{CVaR}_\alpha\Big(\sum_t \gamma^t C(x_t,a_t)\Big) \]

核心机制是 ambiguity budget \(\kappa\)：允许 worst-case 尾部权重在样本间重新分配。本文不是继续 GridWorld 复现，而是将该思想迁移到真实股票/ETF 动态组合。

1.2 代码整体完成了什么

代码实现了一条完整科研流水线，分两个阶段：

阶段目标状态
V1数据 → RCVaR 层验证 → Rolling A/B/C/D → Baseline → RL 初版✅ 完成
V2Risk Measure Learning 框架；ETF10/ETF20/SP30 跨市场消融✅ 完成
\(s_t\) → \(\kappa(s_t)\) → \(\mathrm{RCVaR}_{\alpha,\kappa}\) → 优化 \(w_t\) → 净损失 \(L_{t+1}\) → 回测指标

主贡献定位（V2）：提出并验证状态依赖的风险度量学习——市场越 stress（高波动、高回撤等），\(\kappa(s)\) 越大，RCVaR 对尾部越保守。主实验用 Rolling 优化，不用 RL。

2. 数学模型（完整）

2.1 组合与净损失

\(N\) 个资产，日收益 \(r_t\in\mathbb R^N\)，权重 \(w_t\in\Delta^N\)（单纯形）。组合毛收益 \(R^p_{t+1}=w_t^\top r_{t+1}\)。换手率 \(\|w_t-w_{t-1}\|_1\)，成本率 \(c\)，净收益：

\[ R^{\mathrm{net}}_{t+1} = w_t^\top r_{t+1} - c\|w_t-w_{t-1}\|_1 \]

优化与评价统一使用净损失（不是收益右尾）：

\[ \boxed{L_{t+1}(s_t,w_t) = -w_t^\top r_{t+1} + c\|w_t-w_{t-1}\|_1} \]

2.2 普通 CVaR（经验形式）

给定 \(B\) 个损失样本 \(Z_1,\ldots,Z_B\)，\(\alpha=5\%\) 时：

\[ \mathrm{CVaR}_\alpha(Z) = \max_{q\in\mathcal Q_\alpha}\sum_{i=1}^B q_i Z_i,\quad \mathcal Q_\alpha = \Big\{q\ge0:\ \sum_i q_i=1,\ 0\le q_i\le\frac{1}{\alpha B}\Big\} \]

2.3 鲁棒 CVaR（RCVaR）——本文风险层

\[ \boxed{\mathrm{RCVaR}_{\alpha,\kappa}(Z)=\max_{q\in\mathcal Q_{\alpha,\kappa}}\sum_{i=1}^B q_i Z_i} \] \[ \mathcal Q_{\alpha,\kappa}=\Big\{q\ge0:\ \sum_i q_i=1,\ 0\le q_i\le\frac{\kappa_i}{\alpha B}\Big\} \]

实现：按 \(Z_i\) 从大到小排序，对每个样本分配权重，上限为 \(\kappa_i/(\alpha B)\)，直到总权重为 1。

2.4 退化关系（理论自检）

设定含义
\(\kappa_i=1\)退化为普通 CVaR
\(\kappa_i=K\) 常数Fixed Robust CVaR，等价于更保守的 \(\mathrm{CVaR}_{\alpha/K}\)
\(\kappa_i=\kappa(s_i,w_i)\)状态/动作依赖鲁棒 CVaR（本文创新）

2.5 市场状态与 κ 设计

V2 状态向量（rolling 标准化后）：

\[ s_t = \big(\widetilde{Vol}_t,\ \widetilde{DD}_t,\ \widetilde{Mom}_t,\ \widetilde{Corr}_t\big) \]

主方法 C（manual κ）：

\[ \kappa(s_t) = 1 + \kappa_{\max}\cdot\sigma\big(\beta_1\widetilde{Vol}_t+\beta_2\widetilde{DD}_t+\beta_3\widetilde{Mom}_t+\beta_4\widetilde{Corr}_t\big),\quad \kappa_{\max}=1 \]

故 \(\kappa\in[1,2]\)：市场 stress 越高，尾部样本在 worst-case 分布中可获更高权重 → RCVaR 更保守。

D 模型（消融）在 C 基础上加入组合集中度 \(\mathrm{Conc}(w)=\sum_i w_i^2\)。

2.6 Rolling 组合优化问题

每个调仓日 \(t\)，用过去 \(M=252\) 日历史，求解：

\[ \boxed{w_t^* = \arg\min_{w\in\Delta^N}\ \mathrm{RCVaR}_{\alpha,\kappa}\Big(\{-w^\top r_\tau + c\|w-w_{t-1}\|_1\}_{\tau=t-M}^{t-1}\Big)} \]

权重通过 softmax 参数化保证 \(w\in\Delta^N\)；调仓频率：每月末；样本外测试：2020–2024。

3. 设计逻辑：从 GridWorld 到真实组合

3.1 为什么先做 Rolling，后做 RL

金融 RL 噪声大、训练不稳定。若 RCVaR 层本身有误，RL 结果无法解释。因此执行顺序为：
数据与指标 → RCVaR 层 → Rolling 验证 →（可选）RL

V1/V2 实验表明：Rolling + κ(s) 已能给出可写论文的尾部风险控制结果；当前 RL 仅为快速验证，未超过 Rolling。

3.2 五模型消融的设计意图

模型κ要回答的问题
A1（无鲁棒）baseline：普通 CVaR 组合
B2（固定）固定 ambiguity 是否已有帮助？
C_manualκ(s)状态依赖风险度量是否最优？（主方法）
C_learnedκθ(s)能否从 val 数据学习 θ？
Dκ(s,w)动作依赖是否带来额外 tail 控制，还是噪声？

3.3 跨市场设计（V2）

  • ETF10：主实验，10 只美股 ETF，分散但可解释
  • ETF20：扩展分散度，检验 κ 是否仍有效
  • SP30：29 只蓝筹（akshare 长历史），检验股票组合稳健性

数据：akshare stock_us_daily；划分 train 2010–2017 / val 2018–2019 / test 2020–2024。

7. 结论、局限与论文建议

7.1 可以写进论文的结论

  1. RCVaR 风险层在真实市场有效：Rolling RCVaR 的 CVaR 显著低于 Equal Weight / Min Var。
  2. 状态依赖 κ(s) 是主推方法：ETF10 上 CVaR、MDD、2020 危机损失全面最优；SP30 上 CVaR 比无 κ baseline 低约 10%。
  3. Fixed κ 是强对照：B 在收益-风险折中上优于 A，但 tail 控制略逊于 C_manual。
  4. 消融支持 V2 叙事：去掉 κ（A）尾部恶化；D 未优于 C，action-dependent 引入噪声。

7.2 必须如实报告的局限

  • ETF20：各模型 CVaR 差异 <0.03%，κ 边际收益不明显。
  • learned κ：当前 val 线性学习弱于 manual β，不宜作为主方法。
  • RL：快速训练版未分化，CVaR≈2.75%，不如 Rolling；仅作 future work。
  • 数据：akshare 美股接口；ETF10 有效样本约从 2016 起；SP30 为 29 只（HON 剔除）。

7.3 建议论文结构

  1. Introduction：Ni & Lai robust CVaR → 金融组合迁移
  2. Method：RCVaR + κ(s) risk measure learning + rolling optimization
  3. Experiments：ETF10 主表 + SP30 稳健性 + ETF20 边界讨论
  4. Ablation：A/B/C/D + w/o κ
  5. Conclusion：κ(s) 改善尾部；RL 与非线性 κ 为后续

7.4 Abstract 表述示例

We propose a state-dependent robust CVaR portfolio framework where the ambiguity budget κ(s) adapts to market stress (volatility, drawdown, momentum, correlation). On out-of-sample ETF10 data (2020–2024), our method reduces CVaR5% from 1.67% to 1.63%, maximum drawdown from 20.0% to 17.5%, and COVID crisis loss from 4.2% to 1.8%, compared with standard CVaR optimization. Results on SP30 stocks confirm robust tail-risk improvements.
